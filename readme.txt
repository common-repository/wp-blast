A simple plugin to display your blast in your wordpress site as Yahoo! 360.

== Description ==

WP-Blast allow users to display their blast status in their wordpress site (blog site) and update their blast status though in the Options page.

Their blasts maybe are HTML code or text


== Installation ==

    * Extract the contents of the archives.
    * Upload the wp-blast.php file to your wordpress plugin folder (e.g. (e.g. http://yoursitename.com/wp-content/plugins/).
    * Active it in your Wordpress Admin.
    * Set your blast status in My Blast in your Wordpress Options page.
    * ... that's all.

    Upgrade

    * Extract the contents of the archives.
    * Deactive WP-Blast plugin in your Wordpress Admin.
    * Upload the wp-blast.php file to your wordpress plugin folder and replace the old file. (e.g. (e.g. http://yoursitename.com/wp-content/plugins/).
    * Go to Wordpress Admin and reactive it again!
    * And take a look to the new feature.

    Usage

    * Open your index.php file in your theme folder.
    * Put <?php blastDisplay(); ?> where you want display your blast.
